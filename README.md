# 🎓 EduGate

**A Lightweight Learning Management System (LMS) for School Education**

---

## 📌 Overview

EduGate LMS is a simplified web-based Learning Management System developed as a graduation project for the **Digital Egypt Pioneers Initiative (DEPI)** under the **Full Stack .NET Development Track**.

The platform enables teachers to manage courses, create exams, and track student performance, while allowing students to access educational content and monitor their progress in a structured and user-friendly environment.

---

## 🎯 Project Objectives

* Build a scalable web application using .NET technologies
* Apply full-stack development concepts (Backend + Frontend)
* Design a clean and maintainable system architecture
* Focus on core LMS functionalities

---

## 🧩 Key Features

### 👨‍🏫 Teacher

* Secure Registration & Login
* Create and manage courses
* Generate Class Codes for student enrollment
* Upload course materials
* Create MCQ-based exams
* View student results and performance

---

### 👨‍🎓 Student

* Join courses using Class Code
* Access course content
* Take online exams
* View grades and track progress

---

## 🏗️ System Architecture

* ASP.NET-based Web Application
* Role-Based Access Control (Teacher / Student)
* RESTful API Architecture
* Separation of Concerns (MVC Pattern)

---
